I met a ```pdb``` tutorial on Github; here is the link: https://github.com/spiside/pdb-tutorial

This is the note I made while reading.

hard-coded breakpoints

import pdb; pdb.set_trace()

pdb commands

  1. l(list) [first [,last])
  2. s(tep)
  3. n(ext)
  4. b(reak)
  5. r(eturn)
  6. h(elp)
  7. cl(ear)

8 c(continue)

advanced topics

  1. ! (bang), indicates the next line is Python command and not a ```pdb``` command
  2. Post Mortem: postmortem/ pm

comments:

  1. for a full list of pdb commands: https://docs.python.org/2/library/pdb.html
  2. invoke pdb.py as a script to debug other scrips: python -m pdb myscript.py
